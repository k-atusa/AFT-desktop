목적:
  비밀번호/키 파일 뷰어 및 관리
구조:
  vault는 폴더 하나. 하위에 계정 정보 파일(account.webp / account.png)과 폴더들(N/) 가짐.
  폴더들 안에는 암호화된 파일(N.webp / N.png)들 존재.
  계정 정보는 비빌번호 기반 모드로 보호됨. 안에 옵션 정보, 키 쌍과 암호화된 폴더, 파일에 대한 이름 정보 가짐.
  계정 정보 저장시 백업
  각 파일은 공개키 기반(ecc1/rsa4k)으로 암호화됨.
기능:
  텍스트 파일을 보거나 수정.
  이미지 파일을 봄.
  파일 일부/전체 가져오기/내보내기.
  파일 추가/삭제하기.
  파일 전송(전체/일부)하기.
  비밀번호 변경/키 재생성.

UI:
로그인 {
  볼트 선택 {바로기가 드롭다운, 수동 선택 버튼, 선택 폴더 경로 표시}
  볼트 로그인 {
    메세지 표시
    키 파일 선택, 키 파일 받기, 키 파일 표시
    비밀번호 입력
    로그인 버튼
  }
  새 볼트 생성 {
    폴더 선택 버튼, 폴더 표시
    사진 종류 버튼
    키 종류 버튼
  }
}

메인 {
  기능 {기능 버튼들...}
  트리 뷰 {폴더 구조 표시 (다량의 개수도 보여야 함, 접었다 폈다)}
  내용 보기 {텍스트 편집창 또는 사진}
  세션 {알고리즘, 볼트 경로, 남은 세션, 자동 로그아웃 옵션, 자동 보기 옵션}
}
